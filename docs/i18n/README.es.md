# IrsanAI TPM Agent Forge — Resumen Estratégico (Español)

Este repositorio usa una arquitectura TPM (detección probabilística temprana de anomalías en señales débiles). No es predicción mágica: es detección anticipada basada en contexto, calidad de datos y validación.

## Equivalencia profesional por país (nota)
- España/LatAm: algunas funciones cambian entre nivel nacional, autonómico y municipal.
- Donde un puesto no existe con el mismo nombre, usar equivalencia funcional (mismo objetivo operativo, otro título).
