# IrsanAI TPM Agent Forge — Synthèse Stratégique (Français)

Le TPM détecte des anomalies structurées dans des signaux faibles. Ce n'est pas une prédiction magique, mais une détection probabiliste précoce sous contraintes de qualité des données et de validation.

## Note d'équivalence des métiers
- Les intitulés et responsabilités varient selon les pays et les juridictions.
- En l'absence d'équivalence directe, utiliser l'équivalent fonctionnel.
